Tuning Panel

This panel allow to set the tuning/intonation for all the sounds (oscillator and percussive)

  • Temperament - select the temperament betweern equally (standard equally tempered semitones) and pythagorean (based on simple fractions and lowest osillator)
  • Tuning - select the base tuning between standard 440Hz A note and 8Hz Low C (432Hz A)
